## Prerequisites

* **Rust Toolchain:** Install `rustc` and `cargo` via [rustup.rs](https://rustup.rs/).
* **Dependencies:** The engine utilizes `num-integer` for large value calculations and `rustfft` for FFT operations.

## Build Instructions

To compile the engine with maximum optimizations (recommended for benchmarking):

```bash
cargo build --release
```

## Running

```bash
cargo run --release
```
